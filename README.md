# llm-apps

Base repository to create app with LLM from smartphone.

## 目的 / Purpose

スマートフォンからリポジトリを作成することはできないため、このリポジトリをベースとして各種アプリケーションの制作を依頼します。PR（プルリクエスト）ごとに異なるアプリケーションを作成していくイメージです。

Since it is not possible to create repositories from a smartphone, this repository serves as a base for requesting the creation of various applications. The concept is to create different applications for each PR (Pull Request).

## 使い方 / Usage

1. このリポジトリから新しいブランチを作成
2. LLMを使用してアプリケーションの制作を依頼
3. 各PRで独立したアプリケーションが作成される

1. Create a new branch from this repository
2. Request application development using LLM
3. Each PR creates an independent application
